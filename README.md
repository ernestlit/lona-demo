# Lona Demo

Lona is a social media app created for young people who want to spend their time online in a healthier, more inspiring way. Instead of focusing on endless scrolling and comparison, the app is designed to encourage positive lifestyle habits, personal growth, creativity, wellness, and meaningful connection.

## Why this app exists

Many young users want social platforms that feel uplifting and supportive. Lona brings together the feel of a modern social app with a purpose: helping users build habits around

- mindfulness and self-care
- healthy routines
- motivation and productivity
- positive communities and friendships
- discovery of inspiring content

## Tech stack

- React
- TypeScript
- TanStack Router and TanStack Start
- Vite
- Tailwind CSS
- shadcn/ui components

## Getting started

1. Install dependencies
   ```bash
   bun install
   ```

2. Start the development server
   ```bash
   bun run dev
   ```

## Project structure

- src/routes — app pages such as home, explore, friends, inbox, and profile
- src/components — reusable UI components for the app shell and social features
- src/lib — mock data and helper utilities
